# ClimateTrace Analytics Dashboards

Advanced geospatial emissions tracking and economic decoupling analysis platform for sustainable development.

## Features
- Real-time global CO2 tracker.
- Geospatial impact mapping (Leaflet).
- Algorithmic future scenario modelling.
- Economic-Emission decoupling matrix.

## Tech Stack
- HTML5 / CSS3 (Glassmorphism design)
- JavaScript (ES6)
- Chart.js
- Leaflet.js
- GitHub Actions (CI/CD Deployment)
